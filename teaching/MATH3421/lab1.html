Practical 1: Metropolis-Hastings

1. Random walk Metropolis. Let’s look at simulating from a normal with zero mean and unit variance using a random walk Metropolis algorithm with uniform \(U(-a,a)\) innovations. Copy and paste the following R chunk into R (or Rstudio).

norm<-function(N,a) 
{
 vec <- rep(0,N)
 theta <- 0
 vec[1] <- theta
 for (i in 2:N) 
 {
  innov <- runif(1,-a,a)
  can <- theta + innov
  laprob <- dnorm(can,log=TRUE)-dnorm(theta,log=TRUE)
  u <- runif(1)
  if (log(u) < laprob){ 
   theta <- can
  }
   vec[i] <- theta
  }
 return(vec)
}

#Run for 10k iters with a=1:

normvec <- norm(10000,1)
par(mfrow=c(2,1))
plot(ts(normvec),ylab="Value",xlab="Iteration",main="Trace plot")
hist(normvec,30,xlab="Value",main="Trace plot")
par(mfrow=c(1,1))

(a) Try out the R code above for this simple Metropolis example. See how different choices of \(a\) affect the mixing of the chain. The example code uses \(a=1\) which isn’t too bad. Try \(a=200\) – this gives a chain which is far too "hot". Very few candidate points are accepted, but when they are, they often represent quite considerable moves within the sample space. Also try \(a=0.1\) – this gives a chain which is far too "cold". Most candidates are accepted, but they represent rather conservative moves within the sample space.

2. A simple Metropolis-Hastings independence sampler. Let’s look at simulating from a gamma \(Gamma(a,b)\) distribution using a Metropolis-Hastings independence sampling algorithm with a normal \(N(a/b,a/b^2)\) proposal distribution i.e. with the same mean and variance as the desired Gamma. Copy and paste the following R chunk into R (or Rstudio).

gammaMH<-function(N,a,b) 
{
 mu <- a/b; sig <- sqrt(a/(b*b))
 vec <- rep(0,N)
 theta <- a/b
 vec[1] <- theta
 for (i in 2:N) 
 {
  can <- rnorm(1,mu,sig)
  laprob1 <- dgamma(can,a,b,log=TRUE)-dgamma(theta,a,b,log=TRUE)
  laprob2 <- dnorm(theta,mu,sig,log=TRUE)-dnorm(can,mu,sig,log=TRUE)
  laprob <- laprob1+laprob2
  u <- runif(1)
  if (log(u) < laprob){ 
   theta <- can
  } 
   vec[i] <- theta
  }
 return(vec)
}

vec <- gammaMH(10000,2.3,2.7)
par(mfrow=c(2,1))
plot(ts(vec),ylab="value",xlab="Iteration",main="Trace plot")
hist(vec,30,xlab="Value",main="Histogram")
par(mfrow=c(1,1))
