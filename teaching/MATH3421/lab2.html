Workshop 2: Gibbs sampling

1. Bivariate normal density. Suppose we wish to generate draws of \[\begin{pmatrix} \theta_1\\ \theta_2 \end{pmatrix}\sim \text{N}\left\{\begin{pmatrix}0\\0\end{pmatrix}\,,\, \begin{pmatrix} 1 & \rho \\ \rho & 1 \end{pmatrix}\right\}\] with \(|\rho|<1\). The associated density is \[\pi(\theta_{1},\theta_{2})\propto \exp\left\{-\frac{1}{2(1-\rho^2)}\left(\theta_{1}^{2}+\theta_{2}^{2}-2\rho \theta_{1}\theta_{2}\right)\right\},\quad -\infty <\theta_1,\theta_2<\infty.\] The full conditional distribution for \(\theta_1\) is \[\theta_{1}|\theta_{2}\sim \text{N}\left(\rho \theta_{2},1-\rho^2\right).\] The full conditional distribution for \(\theta_2\) is \[\theta_{2}|\theta_{1}\sim \text{N}\left(\rho \theta_{1},1-\rho^2\right)\,.\] The following R code implements a Gibbs sampler for the bivariate target distribution above.

gibbs<-function(N,rho) 
 {
  mat <- matrix(ncol=2,nrow=N)
  th1 <- 0
  th2 <- 0
  mat[1, ] <- c(th1,th2)
  for (i in 2:N) 
  {
   th1 <- rnorm(1,rho*th2,sqrt(1-rho^2))
   th2 <- rnorm(1,rho*th1,sqrt(1-rho^2))
   mat[i, ] <- c(th1,th2)
  }
  return(mat)
 }
 
out <- gibbs(1000,0.2)
#Trace plots
plot(ts(out[,1]),xlab="Iteration",ylab="theta1")
plot(ts(out[,2]),xlab="Iteration",ylab="theta2")

(b) Visualising the bivariate chain. Consider the following modified gibbs function that returns both the draws of the Markov chain \(\boldsymbol{\theta}^{(0)}, \boldsymbol{\theta}^{(1)},\ldots,\boldsymbol{\theta}^{(N)}\) and all intermediate state values \((\theta_1^{(0)},\theta_2^{(0)})^T, (\theta_1^{(1)},\theta_2^{(0)})^T, (\theta_1^{(1)},\theta_2^{(1)})^T, (\theta_1^{(2)},\theta_2^{(1)})^T, \ldots, (\theta_1^{(N)},\theta_2^{(N)})^T\).

gibbs2<-function(N,rho)
{
  mat <- matrix(ncol=2,nrow=N)
  #Initialise at (-3,-3)^T to demonstrate burn-in
  th1 <- -3
  th2 <- -3
  mat[1, ] <- c(th1,th2)
  state <-c(th1,th2)
  for (i in 2:N)
  {
    th1 <- rnorm(1,rho*th2,sqrt(1-rho^2))
    #store intermediate state
    state <- rbind(state,c(th1,th2))
    th2 <- rnorm(1,rho*th1,sqrt(1-rho^2))
    #store intermediate state
    state <- rbind(state,c(th1,th2))
    mat[i, ] <- c(th1,th2)
  }
  return(list(mat,state))
}

We can overlay the path that the sampler takes on a contour plot representing the bivariate target density. The following two functions evaluate the bivariate target density (up to proportionality) and generate the contour plot.

#Function to evaluate target density (up to proportionality)

dbvn<-function(th1,th2,rho)
{
   return(exp(-1/(2*(1-rho^2))*(th1^2+th2^2-2*rho*th1*th2)))
}

#Function to produce a contour plot of the target

plotDens<-function(rho)
{
  s <- seq(-3,3,0.01)
  mat <- outer(s,s,dbvn,rho)
  contour(s,s,mat,nlevels=15,col="grey",xlab="theta 1",ylab="theta 2")
}

rbivnorm2<-function(N=1000,rho=0.99)
{
  V <- matrix(c(1,rho,rho,1),nrow=2)
  A <- t(chol(V))
  Z <- matrix(rnorm(2*N),nrow=2)

}

We can plot the data via

times <- seq(1851,1962,1)
plot(times,y,lwd=2,main="Coal Disasters",xlab="Year",ylab="Number per year")

Inspection of the plot reveals a reduction in disasters per year over the period. Consequently, the following model was suggested by Carlin et al (Hierarchical Bayesian Analysis of Changepoint Problems, Appl. Statist. 1992): \[\begin{aligned} Y_{i}|\theta &\sim \text{Po}(\theta)\,, \qquad i=1,\ldots ,k\\ Y_{i}|\lambda &\sim \text{Po}(\lambda)\,, \qquad i=k+1,\ldots ,n (=112) \end{aligned}\] This is a standard change point problem in that there is a change in the mean at an unknown point in time.

Given the prior specification, \(\theta\sim \text{Gamma}(a_1,b_{1})\), \(\lambda\sim \text{Gamma}(a_{2},b_{2})\) and \(k\sim\) discrete uniform over \(\{1,2,\ldots ,111\}\), the full conditionals are \[\begin{aligned} \left(\theta|\lambda ,k,\boldsymbol{y}\right)&\sim \text{Gamma}\left(a_{1}+\sum_{i=1}^{k}y_{i}\,,\,b_{1}+k\right)\\ \left(\lambda|\theta ,k,\boldsymbol{y}\right)&\sim \text{Gamma}\left(a_{2}+\sum_{i=k+1}^{n}y_{i}\,,\,b_{2}+(n-k)\right)\\ Pr\left(k=j|\theta,\lambda,\boldsymbol{y}\right)&\propto \exp\left\{j(\lambda-\theta)\right\} (\theta/\lambda)^{\sum_{i=1}^{j}y_{i}}, j\in \{1,2,\ldots ,111\} \end{aligned}\] [You might like to verify these later, as an exercise.]
