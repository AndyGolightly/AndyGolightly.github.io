Question 1 (Bivariate normal density)

Consider the following function for sampling a bivariate normal target (with zero mean, unit marginal variances and correlation \(\rho\)) via Gibbs sampling.

gibbs<-function(N,rho)
 {
  mat <- matrix(ncol=2,nrow=N)
  th1 <- 0
  th2 <- 0
  mat[1, ] <- c(th1,th2)
  for (i in 2:N)
  {
   th1 <- rnorm(1,rho*th2,sqrt(1-rho^2))
   th2 <- rnorm(1,rho*th1,sqrt(1-rho^2))
   mat[i, ] <- c(th1,th2)
  }
  return(mat)
 }
 
out<-gibbs(1000,0.2)
#Trace plots

  1. Visualising the bivariate chain. Consider the following modified gibbs function that returns both the draws of the Markov chain \(\boldsymbol{\theta}^{(0)}, \boldsymbol{\theta}^{(1)},\ldots,\boldsymbol{\theta}^{(N)}\) and all intermediate state values \((\theta_1^{(0)},\theta_2^{(0)})^T, (\theta_1^{(1)},\theta_2^{(0)})^T, (\theta_1^{(1)},\theta_2^{(1)})^T, (\theta_1^{(2)},\theta_2^{(1)})^T, \ldots, (\theta_1^{(N)},\theta_2^{(N)})^T\).
gibbs2<-function(N,rho)
{
  mat <- matrix(ncol=2,nrow=N)
  #Initialise at (-3,-3)^T to demonstrate burn-in
  th1 <- -3
  th2 <- -3
  mat[1, ] <- c(th1,th2)
  state <-c(th1,th2)
  for (i in 2:N)
  {
    th1 <- rnorm(1,rho*th2,sqrt(1-rho^2))
    #store intermediate state
    state <- rbind(state,c(th1,th2))
    th2 <- rnorm(1,rho*th1,sqrt(1-rho^2))
    #store intermediate state
    state <- rbind(state,c(th1,th2))
    mat[i, ] <- c(th1,th2)
  }
  return(list(mat,state))
}

We can overlay the path that the sampler takes on a contour plot representing the bivariate target density. The following two functions evaluate the bivariate target density (up to proportionality) and generate the contour plot.

#Function to evaluate target density (up to proportionality)

dbvn<-function(th1,th2,rho)
{
   return(exp(-1/(2*(1-rho^2))*(th1^2+th2^2-2*rho*th1*th2)))
}

#Function to produce a contour plot of the target

plotDens<-function(rho)
{
  s <- seq(-3,3,0.01)
  mat <- outer(s,s,dbvn,rho)
  contour(s,s,mat,nlevels=15,col="grey",xlab="theta 1",ylab="theta 2")
}

As discussed in lectures and in part (a), high dependence between components (updated by the Gibbs sampler) can lead to poor mixing (slow exporation of the target space). In part (c) we will consider (two) direct sampling approaches for the bivariate normal target.

  1. Direct sampling. We are given the following function.
rbivnorm<-function(N=1000,rho=0.99)
{
 mat <- matrix(ncol=2,nrow=N)
 mat[,1] <- rnorm(N)
 mat[,2] <- rnorm(N,rho*mat[,1],sqrt(1-rho^2))
 return(mat)
}

This function uses the factorisation \[ \pi(\theta_1,\theta_2) = \pi(\theta_1)\pi(\theta_2|\theta_1) \] by first drawing \(\theta_1\) from its marginal followed by \(\theta_2|\theta_1\) from the conditional distribution of \(\theta_2\). The intuition behind the validity of this scheme can be found in the solutions to problems class I.

Here’s another way (that we will cover in lectures) for sampling a bivariate normal density using only (independent) draws from univariate normals.

rbivnorm2<-function(N=1000,rho=0.99)
{
  V <- matrix(c(1,rho,rho,1),nrow=2)
  A <- t(chol(V))
  Z <- matrix(rnorm(2*N),nrow=2)
  return(t(A%*%Z))
}

The intuition is as follows. Let \(\boldsymbol{\theta}\sim N_2(0,\Sigma)\) where \[ \Sigma=\begin{pmatrix} 1 & \rho \\ \rho & 1 \end{pmatrix}. \] We first find a matrix \(A\) such that \(A' A = \Sigma\). This can be obtained via a Cholesky decomposition (for example) of \(\Sigma\). We may then set \[ \tilde{\boldsymbol{\theta}} = A'Z \] where \(\boldsymbol{Z}\sim N_2(0,I_2)\) and \(I_2\) is the \(2\times 2\) identity matrix. It is easily checked that the expectation and variance of \(\tilde{\boldsymbol{\theta}}\) are as expected, and the distribution must be normal since we have a linear combination of (jointly) normal random variables. Alternatively, we can directly derive the density of \(\tilde{\boldsymbol{\theta}}\) using the density of \(\boldsymbol{Z}\).

Suppose that \(a_1=a_2=b_1=b_2=1\). The following R function takes as arguments the required number of draws and the data (vector) and aims to return a matrix whose rows are (dependent) samples from the posterior \(\pi(\theta,\lambda,k|\boldsymbol{y})\).

gibbs<-function(N,data)
{
n <- length(data)
theta <- 1; lambda <- 1; k <- 60  #initial param values
a1 <- 1; a2 <- 1; b1 <- 1; b2 <- 1 #prior hyper-params
probVec <- rep(0,n-1)             #conditional pmf for k
out <- matrix(0,nrow=N,ncol=3)  #store samples here

out[1,] <- c(theta,lambda,k) #store initial sample

for(i in 2:N)
{
 #Update theta
 theta <- rgamma(1,a1+sum(data[1:k]),k+b1)
 #Update lambda
 lambda <- rgamma(1,a2+sum(data[(k+1):n]),n-k+b2)
 #calculate pmf for k
 for(j in 1:(n-1))
 {
  probVec[j] <- exp((lambda-theta)*j)*(theta/lambda)**sum(data[1:j])
 }
 #Update k
 k <- sample(n-1,size=1,prob=probVec)
 
 out[i,] <- c(theta,lambda,k)
}
return(out)
}
