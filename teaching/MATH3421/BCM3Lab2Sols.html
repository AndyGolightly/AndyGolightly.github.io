Question 1 (Bivariate normal density)

Consider the following function for sampling a bivariate normal target (with zero mean, unit marginal variances and correlation \(\rho\)).

gibbs<-function(N,rho)
 {
  mat <- matrix(ncol=2,nrow=N)
  th1 <- 0
  th2 <- 0
  mat[1, ] <- c(th1,th2)
  for (i in 2:N)
  {
   th1 <- rnorm(1,rho*th2,sqrt(1-rho^2))
   th2 <- rnorm(1,rho*th1,sqrt(1-rho^2))
   mat[i, ] <- c(th1,th2)
  }
  return(mat)
 }

The ESS values for the components of the Markov chain produced using \(\rho=0.99\) are small (typically 10-20). Note that we often give a single summary as minimum (over component chains) ESS, with the intuition that the overall efficiency can be summarised by the worst (in terms of ESS) mixing component chain.

rbivnorm<-function(N=1000,rho=0.99)
{
 mat <- matrix(ncol=2,nrow=N)
 mat[,1] <- rnorm(N)
 mat[,2] <- rnorm(N,rho*mat[,1],sqrt(1-rho^2))
 return(mat)
}

The function above uses the factorisation \[ \pi(\theta_1,\theta_2) = \pi(\theta_1)\pi(\theta_2|\theta_1) \] by first drawing \(\theta_1\) from its marginal followed by \(\theta_2|\theta_1\) from the conditional distribution of \(\theta_2\). The intuition behind the validity of this scheme can be found in the solutions to problems class I.

rbivnorm2<-function(N=1000,rho=0.99)
{
  V <- matrix(c(1,rho,rho,1),nrow=2)
  A <- t(chol(V))
  Z <- matrix(rnorm(2*N),nrow=2)
  return(t(A%*%Z))
}

Let \(\boldsymbol{\theta}\sim N_2(0,\Sigma)\) where \[ \Sigma=\begin{pmatrix} 1 & \rho \\ \rho & 1 \end{pmatrix}. \] We first find a matrix \(A\) such that \(A' A = \Sigma\). This can be obtained via a decomposition of \(\Sigma\). We may then set \[ \tilde{\boldsymbol{\theta}} = A'Z \] where \(\boldsymbol{Z}\sim N_2(0,I_2)\) and \(I_2\) is the \(2\times 2\) identity matrix. It is easily checked that the expectation and variance of \(\tilde{\boldsymbol{\theta}}\) are as expected, and the distribution must be normal since we have a linear combination of normal random variables. Alternatively, we can directly derive the density of \(\tilde{\boldsymbol{\theta}}\) using the density of \(\boldsymbol{Z}\).

Suppose that \(a_1=a_2=b_1=b_2=1\). The following R function takes as arguments the required number of draws and the data (vector) and aims to return a matrix whose rows are (dependent) samples from the posterior \(\pi(\theta,\lambda,k|\boldsymbol{y})\).

gibbs<-function(N,data)
{
n <- length(data)
theta <- 1; lambda <- 1; k <- 60  #initial param values
a1 <- 1; a2 <- 1; b1 <- 1; b2 <- 1 #prior hyper-params
probVec <- rep(0,n)             #condiional pmf for k
out <- matrix(0,nrow=N,ncol=3)  #store samples here

out[1,] <- c(theta,lambda,k) #store initial sample

for(i in 2:N)
{
 #Update theta
 theta <- rgamma(1,a1+sum(data[1:k]),k+b1)
 #Update lambda
 lambda <- rgamma(1,a2+sum(data[(k+1):n]),n-k+b2)
 #calculate pmf for k
 for(j in 1:n)
 {
  probVec[j] <- exp((lambda-theta)*j)*(theta/lambda)**sum(data[1:j])
 }
 #Update k
 k <- sample(n,size=1,prob=probVec)
 
 out[i,] <- c(theta,lambda,k)
}
return(out)
}

  1. What if we initialise at k=112? We get an error, as in this case sum(data[(k+1):n]) in the \(\lambda\) update returns “NA” which gets passed to probVec.

We need to remedy this by drawing \(\lambda\) from its prior when \(k=n\).

#Now include fix for k=n

gibbs<-function(N,data)
{
n <- length(data)
theta <- 1; lambda <- 1; k <- 60  #initial param values
a1 <- 1; a2 <- 1; b1 <- 1; b2 <- 1 #prior hyper-params
probVec <- rep(0,n)             #condiional pmf for k
out <- matrix(0,nrow=N,ncol=3)  #store samples here

out[1,] <- c(theta,lambda,k) #store initial sample

for(i in 2:N)
{
 #Update theta
 theta <- rgamma(1,a1+sum(data[1:k]),k+b1)
 #Update lambda
 if(k==n)
 {
  #FCD is the prior
  lambda <- rgamma(a2,b2)
 }else{
  lambda <- rgamma(1,a2+sum(data[(k+1):n]),n-k+b2)
 }
 #calculate pmf for k
 for(j in 1:n)
 {
  probVec[j] <- exp((lambda-theta)*j)*(theta/lambda)**sum(data[1:j])
 }
 #Update k
 k <- sample(n,size=1,prob=probVec)
 
 out[i,] <- c(theta,lambda,k)
}
return(out)
}
