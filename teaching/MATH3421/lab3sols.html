Practical 3 solutions: Bayesian inference via MCMC

Deterministic epidemic model. The following data set consists of the daily number of pupils confined to bed (out of a total of 763) during an influenza outbreak in a boys’ boarding school in Great Britain, instigated by a single pupil (see e.g. BMJ News and Notes, 1978):

No. of infectives \(Y_t\) 1 3 6 25 73 221 294 257 236 189
Day \(t\) 10 11 12 13 14
No. of infectives \(Y_t\) 125 67 26 10 3

(a) Create a length-15 vector y containing the number of infectives and plot the data as a time-series:

y = c(1,3,6,25,73,221,294,257,236,189,125,67,26,10,3)
plot(ts(y,start=0,deltat=1),main="",xlab="Time (days)",ylab="Yt")

A deterministic susceptible-infected-removed (SIR) epidemic model is proposed to describe the dynamics of infectives \(I_t\) (and the unobserved susceptibles, \(S_t\)). This model is given by a coupled system of ordinary differential equations: \[\begin{aligned} \frac{dS_t}{dt}&=-\beta S_t I_t\\ \frac{dI_t}{dt}&= \beta S_t I_t -\gamma I_t\end{aligned}\] where \(\beta\) and \(\gamma\) are infection and removal rates respectively.

The above ODE system can be numerically solved via a simple Euler approximation with time-step \(\Delta t\): \[\begin{aligned} S_{t+\Delta t}&\approx S_t-\beta S_t I_t\Delta t\\ I_{t+\Delta t}&\approx I_t+(\beta S_t I_t -\gamma I_t)\Delta t\end{aligned}\] That is, starting with a value \((S_0,I_0)'\), we apply the above recursively to give \((S_{\Delta t},I_{\Delta t})'\), \((S_{2\Delta t},I_{2\Delta t})'\) etc.

The following R function takes arguments a parameter vector \((\beta,\gamma)'\) (coded as param), the time-step \(\Delta t\) (coded as dt), an initial value \(x_0=(S_0,I_0)'\) (coded as x0) and an end time \(T\) (coded as endT). It then returns a single numerical simulation of the ODE system.

odesim=function(param,dt,x0,endT)
{
 n = endT/dt
 x = matrix(0,nrow=n+1,ncol=2)
 x[1,] = x0 
 for (i in 2:(n+1)) 
 {
  #right hand side of ODE system
  RHS1 = -param[1]*x[i-1,1]*x[i-1,2]
  RHS2 = param[1]*x[i-1,1]*x[i-1,2]-param[2]*x[i-1,2]
  #time step ODE system
  x[i,] = x[i-1,] + c(RHS1,RHS2)*dt
 }
 return(x)
}

Each observed data point \(Y_t\) is linked to the ODE system via the following statistical model: \[Y_t|I_t,\sigma \overset{\textrm{indep}}{\sim} N(I_t,\sigma^2), \qquad t=0,1\ldots,14.\] Using the data above, we would like to learn about the infection and removal rates, the parameter \(\sigma\) that allows for the discrepancy between the ODE model and data, and the unobserved susceptible process.

As the parameters must be strictly positive, let \(\boldsymbol{\theta}=(\theta_1,\theta_2,\theta_3)'=(\log\beta,\log\gamma,\log\sigma)'\) and consider the posterior density \[\pi(\boldsymbol{\theta}|\boldsymbol{y})\propto \pi(\theta_1)\pi(\theta_2)\pi(\theta_3)f(\boldsymbol{y}|\boldsymbol{\theta})\] which assumes an independent prior specification. In particular, we will take \(\theta_i\overset{\textrm{indep}}{\sim} N(0,1)\), \(i=1,2,3\). The following function returns the evaluation of the log-likelihood \(\log f(\boldsymbol{y}|\boldsymbol{\theta})\) at the value theta.

loglike=function(theta,data,dt,x0)
{
  n = length(data)
  incr = 1/dt
  out = odesim(exp(theta),dt,x0,n-1)
  out = out[1+(0:(n-1))*incr,2] #assume obs are infecteds
  return(sum(dnorm(data,out,exp(theta[3]),log=TRUE)))
}

The likelihood is \[f(\boldsymbol{y}|\boldsymbol{\theta})=\prod_{t=0}^{14} N(y_t; i_t,\sigma^2).\] The key to understanding this expression as a function of \(\boldsymbol{\theta}\) is to note that \(i_t\) is a deterministic function of \(\beta\) and \(\gamma\). Now, we just need the output of the ODE solver at the observation times. This is the purpose of the second to last line in the code above.

(b) Write an R function logprior that takes a single argument theta and returns the evaluation of the log-prior \(\log \pi(\boldsymbol{\theta})\).

logprior=function(param)
{
 return(sum(dnorm(param,log=TRUE)))
}

(c) The following complete R function implements a Metropolis-Hastings sampler with target \(\pi(\boldsymbol{\theta}|\boldsymbol{y})\) and a random walk proposal \[\boldsymbol{\theta}^*|\boldsymbol{\theta}\sim N(\boldsymbol{\theta},\Sigma)\] where \(\Sigma\) is a tuning matrix chosen by the practitioner – you!

#You will need the MASS package
#Uncomment the second line below if already installed!
#install.packages("MASS")
#library(MASS)

MH=function(N,data,dt,Sigma,init,x0)
{
  theta.mat = matrix(0,nrow=N,ncol=3)
  theta.mat[1,] = init
  curr = init
  count = 0
  for (j in 2:N) 
    {
    can = mvrnorm(1,curr,Sigma) #proposal 
    #Evaluate log acceptance probability
    laprob = logprior(can)-logprior(curr)+
    loglike(can,data,dt,x0)-loglike(curr,data,dt,x0)
    if (log(runif(1)) < laprob) #Accept with the correct prob.
    {
      curr = can; #chain moves
      count = count + 1 #add one to acceptance counter
    }
    theta.mat[j,] = curr
  }
  print(count/(N-1)) # empirical acceptance prob
  return(theta.mat)
}

Run the sampler for \(N=1000\) iterations for the data set \(y\), with a time step of 0.1, a tuning matrix of \(\Sigma=\textrm{diag}\{0.01,0.01,0.01\}\), and initial value of \(\boldsymbol{\theta}^{(0)}=(-6,\log 0.5, 2.5)'\) and \(x_0=(762,1)'\):

out=MH(1000,y,0.1,diag(rep(0.001,3)),c(-6,log(0.5),2.5),c(762,1))

Examine trace plots of the sampler output.

#Trace plots
plot(ts(out))

The trace plot for \(\log\sigma\) suggests that a much larger innovation variance is needed for this component. We will remedy in part (d).

(e) The basic reproduction number is given by \[R_0 = \frac{\beta}{\gamma}\times N_{pop}\] where \(N_{pop}=763\) is the the population size. It is, loosely speaking, the expected number of secondary infections resulting from a single infective in a population of susceptibles. Using the sampler output from (d), produce a histogram of sampled \(R_0\) values and a equi-tailed 95% posterior credible interval for \(R_0\).

R0=out3[,1]/out3[,2]*763
hist(R0,xlab="R0")
quantile(R0,c(0.025,0.975))

Suggests around 3.5-4 secondary infections from a single primary infective. Highly transmissible!

(f) Let \(S_{(0,14]}\) denote the susceptible process over \((0,14]\). This is a deterministic function of the parameters \(\boldsymbol{\theta}\) and the initial condition \(X_0\). That is, given \(\boldsymbol{\theta}\) and \(X_0\), a numerical solution of the SIR ODE system can be obtained, giving \(S_{(0,14]}\). We then average over parameter uncertainty with the following code:

N = 5000
endT = 14
dt = 0.1
S.mat = matrix(0,nrow=endT/dt+1,ncol=N)
for(i in 1:N)
{
 S.mat[,i] = odesim(exp(out2[i,]),dt,c(762,1),endT)[,1]
}

This gives S.mat as a matrix with \(N\) columns with each column containing a numerical realisation of \(S_{[0,14]}\) over \([0,14]\).

The following code computes the mean, upper and lower 2.5 percentiles of the \(S_t\) realisations at times \(0,0.1,0.2,0.3,\ldots,14\):

S.mean = apply(S.mat,1,mean)
S.lq = apply(S.mat,1,quantile,0.025)
S.uq = apply(S.mat,1,quantile,0.975)

Hence, give a time series plot of the mean of the susceptible process and overlay a 95% credible interval. The resulting plot is a summary of the predictive distribution of the susceptible process.

plot(ts(S.mean,start=0,deltat=0.1),ylim=c(0,800))
lines(ts(S.lq,start=0,deltat=0.1),col=2)
lines(ts(S.uq,start=0,deltat=0.1),col=2)

This gives a summary of the predictive distribution of the susceptible process. For a reminder on predictive distributions, see Section 5.6 (page 148) of your Statistical Inference II notes. There is also some useful information in our own problems class I material.

Bonus 2: the statistical model \[Y_t|I_t,\sigma \overset{\textrm{indep}}{\sim} N(I_t,\sigma^2), \qquad t=0,1\ldots,14.\] assumes that the observation error / discrepancy between the observations and latent infective process has the same variance irrespective of the latent infected state.

There are various possibilities here. We could assume that \[Y_t|I_t,\sigma \overset{\textrm{indep}}{\sim} N(I_t,\sigma^2 I_t), \qquad t=0,1\ldots,14\] which is rather like a normal approximation to a Poisson distribution, with the parameter \(\sigma^2\) allowing for a decoupling of the mean and variance. For a strictly positive observation model, how about starting with \[\log Y_t |I_t,\sigma \overset{\textrm{indep}}{\sim} N(\log I_t,\sigma^2), \qquad t=0,1\ldots,14\] to give \[Y_t|I_t,\sigma \overset{\textrm{indep}}{\sim} logN(\log I_t,\sigma^2), \qquad t=0,1\ldots,14,\] where \(logN(,)\) denotes the lognormal distribution.
