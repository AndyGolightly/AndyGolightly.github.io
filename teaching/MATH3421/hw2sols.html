  1. (a) From the R code, and in particular, the (logarithm of the) acceptance probability, we identify \[\pi(x)\propto \exp(-|x|), \quad x\in \mathbb{R}.\] Let \(k\) denote the normalising constant. Then \[\begin{aligned} k&=\int_{-\infty}^0 \exp(x) + \int_{0}^\infty \exp(-x)\\ &=[\exp(x)]_{-\infty}^0 - [\exp(-x)]_0^{\infty}\\ &=2. \end{aligned}\] Hence, \[\pi(x)= \frac{1}{2}\exp(-|x|), \quad x\in \mathbb{R}.\]

    (b) The proposal is a random walk with normal innovations, hence \(q(x^*|x)=N(x^*; x,\sigma^2)\). This density can be easily given explicitly.

  2. (a)

    (i) First note that the proposal density is \(q(\theta^*)=2/\pi\) with the same support as \(f(\theta)\). Hence, the ratio of proposal densities in the acceptance probability will be 1.

    Now, the steps required are:

    1. Initialise the chain somewhere in the support of \(f(\theta)\) e.g. \(\pi/4\). Set the iteration counter \(j=1\).

    2. Propose \(\theta^*\sim U(0,\pi/2)\).

    3. Evaluate the acceptance probability \(\alpha(\theta^{*}|\theta^{(j-1)})\) of the proposed move, \[\alpha(\theta^{*}|\theta)=\textrm{min}\left\{ 1, \frac{\sin(\theta^*)}{\sin(\theta^{(j-1)})} \right\}.\]

    4. Put \(\theta^{(j)}=\theta^{*}\) with probability \(\alpha(\theta^{*}|\theta^{(j-1)})\); otherwise put \(\theta^{(j)}=\theta^{(j-1)}\).

    5. If \(j=N\) stop, otherwise put \(j\) to \(j+1\) and go to step 2.

    (ii) We have that \[\begin{aligned} \mathbb{P}(\textrm{move}|\theta)&= \int_{0}^{\pi/2} \alpha(\phi|\theta) q(\phi)d\phi\\ &=\int_{0}^{\pi/2} \min\left\{1,\frac{\sin\phi}{\sin\theta}\right\}\times \frac{2}{\pi}d\phi\\ &=\frac{2}{\pi}\int_{0}^{\theta}\frac{\sin\phi}{\sin\theta}d\phi +\frac{2}{\pi}\int_{\theta}^{\pi/2} d\phi \end{aligned}\] where the first term in the last line is the case that \(\phi<\theta\) and the second is for \(\phi>\theta\). Hence, we obtain \[\begin{aligned} \mathbb{P}(\textrm{move}|\theta)&= \frac{2}{\pi}\left[-\frac{\cos\phi}{\sin\theta}\right]_0^\theta + \frac{2}{\pi}\left[\phi\right]_\theta^{\pi/2}\\ &=\frac{2}{\pi} \left(\frac{1-\cos\theta}{\sin\theta}-\theta\right) + 1. \end{aligned}\]

    (iii) We know that \[\begin{aligned} \mathbb{P}(\textrm{move}) &= \int \mathbb{P}(\textrm{move}|\theta)f(\theta)d\theta\\ &=\frac{2}{\pi}\int_{0}^{\pi/2}(1-\cos\theta)d\theta - \frac{2}{\pi}\int_{0}^{\pi/2}\theta\sin(\theta) d\theta +\int_{0}^{\pi/2}\sin(\theta) d\theta . \end{aligned}\] Note that the second term requires integration by part, which gives \[\int_{0}^{\pi/2}\theta\sin(\theta) d\theta = 1.\] Hence, \[\begin{aligned} \mathbb{P}(\textrm{move}) &= \left[\frac{2}{\pi}(\theta-\sin\theta)\right]_{0}^{\pi/2} -\frac{2}{\pi} +\left[-\cos\theta \right]_{0}^{\pi/2}\\ &=2-\frac{4}{\pi}\\ &=0.727. \end{aligned}\] The MH scheme is an independence sampler. Ideally, the overall acceptance rate should be as close to 1 as possible. The overall rate of around 0.7 is not terrible, but it would better to directly generate draws from the target using the inverse cdf method.

    (iv) The following R code will implement the sampler and estimate the overall acceptance rate:

    Q2a=function(N=1000,theta0=pi/4)
{
 theta=rep(0,N)  # Store theta values here
 theta[1]=theta0 # Initialise
 count=0 # count acceptances
 for(i in 2:N)
 {
  can=runif(1,0,pi/2)
  aprob=sin(can)/sin(theta[i-1])
  if(runif(1)<aprob)
  {
   theta[i]=can # Store candidate value
   count=count+1
  }else{
   theta[i]=theta[i-1] # Else store current value
  }
 }
print(count/(N-1))
return(theta)
} 

out=Q2a()

    (b)

    (i) The steps required are:

    1. Initialise the chain somewhere in the support of \(f(\theta)\) e.g. \(\pi/4\). Set the iteration counter \(j=1\).

    2. Propose \(\theta^*\sim q(\cdot)\) (e.g. using inverse sampling).

    3. Evaluate the acceptance probability \(\alpha(\theta^{*}|\theta^{(j-1)})\) of the proposed move, \[\alpha(\theta^{*}|\theta)=\textrm{min}\left\{ 1, \frac{\sin(\theta^*)}{\sin(\theta^{(j-1)})}\times \frac{\theta^{(j-1)}}{\theta^*} \right\}.\]

    4. Put \(\theta^{(j)}=\theta^{*}\) with probability \(\alpha(\theta^{*}|\theta^{(j-1)})\); otherwise put \(\theta^{(j)}=\theta^{(j-1)}\).

    5. If \(j=N\) stop, otherwise put \(j\) to \(j+1\) and go to step 2.

    (ii) As in Q2(a), we write \[\begin{aligned} \mathbb{P}(\textrm{move}|\theta)&= \int_{0}^{\pi/2} \alpha(\phi|\theta) q(\phi)d\phi\\ &=\int_{0}^{\pi/2} \min\left\{1,\frac{\sin\phi}{\sin\theta}\times \frac{\theta}{\phi}\right\}\frac{8\phi}{\pi^2} d\phi. \end{aligned}\] Note that \(\sin(\phi)/\phi\) is decreasing in \(\phi\) over \([0,\pi/2]\). Hence, for \(\phi<\theta\), \(\sin(\phi)/\phi>\sin(\theta)/\theta\). Thus we obtain \[\begin{aligned} \mathbb{P}(\textrm{move}|\theta)&=\int_0^\theta \frac{8\phi}{\pi^2}d\phi+ \int_\theta^{\pi/2} \frac{8\theta\sin\phi}{\pi^2\sin\theta}d\phi\\ &= \left[\frac{4\phi^2}{\pi^2}\right]_0^\theta + \left[-\frac{8\theta\cos\phi}{\pi^2\sin\theta}\right]_\theta^{\pi/2}\\ &=\frac{4\theta^2}{\pi^2}+\frac{8\theta\cos\theta}{\pi^2\sin\theta}. \end{aligned}\]

    (iii) We need \[\begin{aligned} \mathbb{P}(\textrm{move}) &= \int \mathbb{P}(\textrm{move}|\theta)f(\theta)d\theta\\ &=\int_{0}^{\pi/2}\left(\frac{4\theta^2}{\pi^2}+\frac{8\theta\cos\theta}{\pi^2\sin\theta}\right)\sin(\theta) d\theta\\ &=\int_{0}^{\pi/2} \frac{4\theta^2\sin\theta}{\pi^2} d\theta + \int_{0}^{\pi/2}\frac{8\theta\cos\theta}{\pi^2} d\theta. \end{aligned}\] Now, upon integrating by part, we obtain \[\int_0^{\pi/2}\theta\cos(\theta)d\theta=\frac{\pi}{2}-1.\] Similarly, \[\int_0^{\pi/2}\theta^2\sin(\theta)d\theta=2\int_0^{\pi/2}\theta\cos(\theta)d\theta = \pi-2.\] Hence \[\begin{aligned} \mathbb{P}(\textrm{move})&= \frac{4}{\pi^2}(\pi-2) + \frac{8}{\pi^2}(\frac{\pi}{2}-1)\\ &=\frac{8(\pi-2)}{\pi^2}\\ &=0.93. \end{aligned}\]

    (iv) The following R code implements the sampler and prints the empirical acceptance rate:

    Q2b=function(N=1000,theta0=pi/4)
{
 theta=rep(0,N)  # Store theta values here
 theta[1]=theta0 # Initialise
 count=0 # count acceptances
 for(i in 2:N)
 {
  can=sqrt(runif(1))*pi/2 #inverse sampling method
  aprob=sin(can)*theta[i-1]/(sin(theta[i-1])*can)
  if(runif(1)<aprob)
  {
   theta[i]=can # Store candidate value
   count=count+1
  }else{
   theta[i]=theta[i-1] # Else store current value
  }
 }
print(count/(N-1))
return(theta)
} 

out=Q2b()

  3. The Markov chain is NOT irreducible. If the chain is initialised in \([0,1]\) it will stay in \([0,1]\). If it is initialised in \([2,3]\) it will stay in \([2,3]\). (Note that any proposed moves outside of these regions will be rejected, as the target density will evaluate to 0 at these proposals. Note also that if the current value is 1, the proposal is \(\text{U}\)(0,2), which can never return exactly the value 2, due to the corresponding random variable being continuous.)

    Going a little further, let \(\mathcal{X}_1=[0,1]\), \(\mathcal{X}_2=[2,3]\) and \(\mathcal{X}=\mathcal{X}_1\cup\mathcal{X}_2\). WLOG, suppose that \(\theta^{(0)}=\theta\in \mathcal{X}_1\). We require (for \(\mu\)-irreducibility, with \(\mu\) given by the target) \(\mathbb{P}(\theta^{(n)}\in A|\theta)>0\) for some positive integer \(n\) and for all \(A\subseteq \mathcal{X}\). Now take \(A=\mathcal{X}_2\). For \(\mathbb{P}(\theta^{(n)}\in \mathcal{X}_2|\theta)>0\), we require \(\theta^*\in \mathcal{X}_2\), followed by acceptance, where \(\theta^*=\theta^{(n-1)}+\omega\) and \(\omega\sim U(-\delta,\delta)\). If we assume \(\theta^{(n-1)}\in \mathcal{X}_1\), then it should be clear that for \(\delta\leq 1\), \(\theta^*\) is restricted to \(\mathcal{X}_1\). Alternatively, assume that \(\theta^{(n-1)}\in \mathcal{X}_2\). However, this assumption implies the existence of a previously proposed \(\theta\) value in \(\mathcal{X}_2\), given a current \(\theta\) value in \(\mathcal{X}_1\), which contradicts the previous scenario. In other words, there must have been (at least) one transition from \(\mathcal{X}_1\) to \(\mathcal{X}_2\). WLOG, consider the transition kernel \(p(\theta^{(1)}|\theta^{(0)}=\theta)\) where \(\theta\in\mathcal{X}_1\). Conditional on the chain moving, the support of the resulting transition density is (max\((0,\theta-\delta)\), min\((1,\theta+\delta)\)). Hence, \(p(\theta^{(1)}|\theta^{(0)}=\theta)\) is 0 over \(\mathcal{X}_2\).

    Drawing the picture of the target density (being careful to recognise that the density is 0.5 over both regions) and proposal density for \(\delta\leq 1\) and a particular \(\theta\) will also help here.
