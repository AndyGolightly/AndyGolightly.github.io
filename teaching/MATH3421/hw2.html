Tutorial questions 2

  1. Consider the following R function which implements a Metropolis-Hastings sampler:

    > ## Tutorial Question 1 code
> 
> MH=function(N=1000,sigma=1,x0=0)
+ {
+  x=rep(0,N)
+  x[1]=x0
+  for(i in 2:N)
+  {
+   y=rnorm(1,x[i-1],sigma)
+   lalpha=abs(x[i-1])-abs(y)
+   if(log(runif(1))<lalpha)
+   {
+    x[i]=y
+   }else{
+    x[i]=x[i-1]
+   }
+  }
+ return(x)
+ }

    (a) What is the target density, \(\pi(x)\) with \(x\in \mathbb{R}\), of this sampler?

    (b) What is the proposal density, \(q(x^*|x)\), of this sampler?

  2. Consider the following target density: \[f(\theta) = \begin{cases}\sin(\theta)&\textrm{when\ }0\leq \theta\leq\pi/2\\ 0&\textrm{otherwise.}\end{cases}\]

    (a)

    (i) Give the steps of a Metropolis-Hastings independence sampler with target \(f(\theta)\) based on a \(U(0,\pi/2)\) proposal.

    (ii) If the chain is currently at \(\theta\), what is the probability that the chain will move? (iii) When the chain reaches equilibrium, what is the overall acceptance probability? (iv) Compare this probability with an empirical estimate from the sampler, initialised at \(\theta^{(0)}=\pi/4\) and run for 1000 iterations.

    (b)

    (i) Give the steps of a Metropolis-Hastings independence sampler with target \(f(\theta)\) based on a proposal with density \[q(\theta) = \begin{cases}\frac{8\theta}{\pi^2}&\textrm{for\ }0\leq \theta\leq\pi/2,\\ 0&\textrm{otherwise}.\end{cases}\]

    (ii) If the chain is currently at \(\theta\), what is the probability that the chain will move?

    (iii) When the chain reaches equilibrium, what is the overall acceptance probability?

    (iv) Compare this probability with an empirical estimate from the sampler, initialised at \(\theta^{(0)}=\pi/4\) and run for 1000 iterations.

    (v) Which proposal mechanism ((a)(i) or (b)(i)) do you prefer?
