Solutions to Topic 5 Questions

  1. (a) Using the hint, \[\begin{aligned} \mathrm{Var}_g[w_t(X_t)]&= \mathbb{E}_g[w_t(X_t)^2]-\left\{\mathbb{E}_g[w_t(X_t)] \right\}^2\\ &=\left(\int \frac{p(x_t|x_{t-1})p(y_t|x_t)}{g(x_t|x_{t-1})} \right)^2 g(x_t|x_{t-1})dx_t - \left[\int p(y_t|x_t)p(x_t|x_{t-1})dx_t\right]^2\\ &=\left(\int \frac{[p(x_t|x_{t-1})p(y_t|x_t)]^2}{g(x_t|x_{t-1})}dx_t \right) - [p(y_t|x_{t-1})]^2\end{aligned}\] as required.

    (b) We have \(g(x_t|x_{t-1})=p(x_t|x_{t-1},y_t)\). First note that \[\begin{aligned} p(x_t|x_{t-1},y_t)&= \frac{p(x_t,y_t|x_{t-1})}{p(y_t|x_{t-1})}\\ &=\frac{p(x_t|x_{t-1})p(y_t|x_t)}{p(y_t|x_{t-1})}.\end{aligned}\] Now substitute the latter expression into the first term in \(\mathrm{Var}_g[w_t(X_t)]\) to obtain \[\begin{aligned} \mathrm{Var}_g[w_t(X_t)]&=\int p(y_t|x_{t-1})^2 \frac{p(x_t|x_{t-1})p(y_t|x_t)}{p(y_t|x_{t-1})}- [p(y_t|x_{t-1})]^2\\ &=p(y_t|x_{t-1})\int p(x_t|x_{t-1})p(y_t|x_t)dx_t - [p(y_t|x_{t-1})]^2\\ &=[p(y_t|x_{t-1})]^2 - [p(y_t|x_{t-1})]^2\\ &=0\end{aligned}\] as required.

  2. (a) Something like the following will do the job:

    1. Set \(x_0=0\) with probability 0.5 and \(x_0=1\) otherwise. Set \(t=1\).

    2. Set \(x_t=1-x_{t-1}\) with probability \(\epsilon\) and \(x_t=x_{t-1}\) otherwise.

    3. Draw \(y_t\sim N(x_t,1)\).

    4. Set \(t:=t+1\) and go to step 2.

    (b) Note that \(X_t\) is discrete, taking values in the set \(\{x_{t-1},1-x_{t-1}\}\). Hence \(p(x_t|x_{t-1},y_t)\) is the corresponding probability mass function. We have that \[p(x_t|x_{t-1},y_t)\propto p(x_t|x_{t-1})p(y_t|x_t).\] Now suppose \(x_{t-1}=x\). Then, \[p(X_t=x|x_{t-1},y_t)\propto (1-\epsilon)N(y_t;x,1).\] Similarly, \[p(X_t=1-x|x_{t-1},y_t)\propto \epsilon N(y_t;1-x,1).\] The normalising constant in both probabilities is simply the sum of the two expressions above.

    (c) We have \(x_{t-1}=0\). We require \[\begin{aligned} p(X_t=1-x_{t-1}|x_{t-1},y_t) &= p(X_t=x_{t-1}|x_{t-1},y_t) \\ \Rightarrow \epsilon\exp\left(-\frac{1}{2}(y_t-1)^2\right) &= (1-\epsilon)\exp\left(-\frac{1}{2}y_t^2 \right)\\ \Rightarrow y_t &= \frac{1}{2}-\log\left(\frac{\epsilon}{1-\epsilon} \right).\end{aligned}\]

  3. (a) We have that \(X_{t-1}|y_{0:t-1}\sim N(m_{t-1}, V_{t-1})\). Writing the second equation in the model definition as \[X_t=X_{t-1}+\epsilon_t, \quad \epsilon_t\overset{\textrm{indep}}{\sim} N(0,1)\] and using properties of linear combinations of (jointly) Gaussian random variables gives \[X_t|y_{0:t-1}\sim N(m_{t-1},V_{t-1}+1).\]

    (b) Now write the third equation in the model definition as \[Y_t=X_t+\omega_t,\quad \omega_t\overset{\textrm{indep}}{\sim} N(0,1)\] to give \[Y_t|y_{0:t-1}\sim N(m_{t-1},V_{t-1}+2).\] We now have the marginal distributions of \(X_t|y_{0:t-1}\) and \(Y_t|y_{0:t-1}\). We just need the covariance between \(X_t\) and \(Y_t\), conditional on \(y_{0:t-1}\). We have that, upon dropping the explicit dependence on \(y_{0:t-1}\) \[\begin{aligned} \textrm{Cov}(Y_t,X_t)&= \textrm{Cov}(X_t+\omega_t,X_t)\\ &=\mathrm{Var}(X_t)\\ &=V_{t-1}+1.\end{aligned}\] Hence, \[\begin{pmatrix}X_t \\Y_t \end{pmatrix}\big | y_{0:t-1} \sim N\left\{ \begin{pmatrix} m_{t-1} \\ m_{t-1} \end{pmatrix}\,,\, \begin{pmatrix} V_{t-1}+1 & V_{t-1}+1 \\ V_{t-1}+1 & V_{t-1}+2 \end{pmatrix} \right\}.\]

    (c) Using the hint, \(X_t|y_{0:t}\sim N(\mu,\Sigma)\) where \[\begin{aligned} \mu&=m_{t-1}+\frac{V_{t-1}+1}{V_{t-1}+2}(y_t-m_{t-1}),\\ \Sigma&=(V_{t-1}+1)\left(1-\frac{V_{t-1}+1}{V_{t-1}+2}\right).\end{aligned}\]

    The point of this question is that for linear Gaussian hidden Markov models such as the one considered here, the filtering distribution is tractable and can be updated recursively. These steps give an algorithm known in a more general setting as the Kalman filter.
