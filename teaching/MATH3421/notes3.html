3 The Gibbs sampler

Constructing a suitable proposal density \(q(\boldsymbol{\theta}^*|\boldsymbol{\theta})\) may be difficult, especially when the target \(\pi(\boldsymbol{\theta})\) is high dimensional. Sometimes, it is more natural to update subsets of components of \(\boldsymbol{\theta}\) in a single Metropolis-Hastings step, with steps for these different subsets combined to give a single algorithm. Moreover, for many problems of interest, the corresponding full conditional distributions (FCDs) associated with these subsets of components may be available for sampling from. This leads to an MCMC technique known as the Gibbs sampler.

3.1 Component-wise and block-wise transitions

Suppose that \(\boldsymbol{\theta}\) can be decomposed into \((\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)'\) and that, for the moment, \(\boldsymbol{\theta}_B\) is fixed and known. We therefore wish to generate samples from a target density \(\pi(\boldsymbol{\theta}_A|\boldsymbol{\theta}_B)\) using Metropolis-Hastings with a proposal \[q(\boldsymbol{\theta}_A^*|\boldsymbol{\theta}_A):=q(\boldsymbol{\theta}_A^*|\boldsymbol{\theta}_A,\boldsymbol{\theta}_B).\] The MH acceptance probability is \(\min\{1,R\}\) where \[\begin{aligned} R(\boldsymbol{\theta}^{*}_A|\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)&= \frac{\pi(\boldsymbol{\theta}^{*}_A|\boldsymbol{\theta}_B)\,q(\boldsymbol{\theta}_A|\boldsymbol{\theta}_A^{*})}{\pi(\boldsymbol{\theta}_A|\boldsymbol{\theta}_B)\,q(\boldsymbol{\theta}_A^{*}|\boldsymbol{\theta}_A)}\\ &=\frac{\pi(\boldsymbol{\theta}^{*}_A,\boldsymbol{\theta}_B)/\pi(\boldsymbol{\theta}_B)}{\pi(\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)/\pi(\boldsymbol{\theta}_B)}\times \frac{q(\boldsymbol{\theta}_A|\boldsymbol{\theta}_A^{*})}{q(\boldsymbol{\theta}_A^{*}|\boldsymbol{\theta}_A)}\\ &=\frac{\pi(\boldsymbol{\theta}^{*}_A,\boldsymbol{\theta}_B)}{\pi(\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)}\times \frac{q(\boldsymbol{\theta}_A|\boldsymbol{\theta}_A^{*})}{q(\boldsymbol{\theta}_A^{*}|\boldsymbol{\theta}_A)}.\end{aligned}\] Hence, we may use the full target in the acceptance probability; the only difference is that some of the parameters, \(\boldsymbol{\theta}_B\), have not changed. If the update is accepted then the current state of the chain becomes \((\boldsymbol{\theta}_A^*,\boldsymbol{\theta}_B)'\) otherwise it is \((\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)'\).

Now suppose that \(\boldsymbol{\theta}\) is in its stationary distribution. Then, its density is \(\pi(\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)=\pi(\boldsymbol{\theta}_B)\pi(\boldsymbol{\theta}_A|\boldsymbol{\theta}_B)\). If we apply the above MH step to \(\boldsymbol{\theta}_A\) then:

  • By design of the MH step, the conditional distribution \(\pi(\boldsymbol{\theta}_A|\boldsymbol{\theta}_B)\) is preserved.

  • We have not altered \(\boldsymbol{\theta}_B\) so its distribution \(\pi(\boldsymbol{\theta}_B)\) is preserved.

Hence, \((\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)'\sim \pi(\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)\).

Problem: If only \(\boldsymbol{\theta}_A\) is updated, then the Markov chain would be reducible, so it would never converge to \(\pi(\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)\) in the first place.

Solution: We alternate updates to \(\boldsymbol{\theta}_A\) and \(\boldsymbol{\theta}_B\), with the latter using some proposal density \(q(\boldsymbol{\theta}_B^*|\boldsymbol{\theta}_B):=q(\boldsymbol{\theta}_B^*|\boldsymbol{\theta}_A,\boldsymbol{\theta}_B)\).

This approach applies more generally, with \(\boldsymbol{\theta}\) split into three or more vectors e.g. \(\boldsymbol{\theta}_A\), \(\boldsymbol{\theta}_B\), \(\boldsymbol{\theta}_C\),... This is called a block update. Now recall that \(\boldsymbol{\theta}=(\theta_1,\theta_2,\ldots,\theta_d)'\) and denote the FCD for the \(i\)th component of \(\boldsymbol{\theta}\) by \[\pi(\theta_i|\boldsymbol{\theta}_{-i}):=\pi(\theta_i|\theta_{1},\ldots,\theta_{i-1},\theta_{i+1},\ldots,\theta_{d}).\] An algorithm that cycles through MH steps for all components in turn is as follows:

  1. Initialise the state of the chain to \(\boldsymbol{\theta}^{(0)}=(\theta_1^{(0)},\ldots,\theta_d^{(0)})'\). Initialise the iteration counter to \(j=1\).

  2. Obtain a new value \(\boldsymbol{\theta}^{(j)}\) from \(\boldsymbol{\theta}^{(j-1)}\) by successive generation of values

    • \(\theta_1^{(j)}\sim \pi(\theta_1|\theta_{2}^{(j-1)},\ldots,\theta_{d}^{(j-1)})\) using a Metropolis–Hastings step with proposal distribution \(q_1(\phi_1|\theta_1^{(j-1)})\)

    • \(\theta_2^{(j)}\sim \pi(\theta_2|\theta_{1}^{(j)},\theta_{3}^{(j-1)},\ldots,\theta_{d}^{(j-1)})\) using a Metropolis–Hastings step with proposal distribution \(q_2(\phi_2|\theta_2^{(j-1)})\)

    • \(\vdots\)

    • \(\theta_d^{(j)}\sim \pi(\theta_d|\theta_{1}^{(j)},\ldots,\theta_{d-1}^{(j)})\) using a Metropolis–Hastings step with proposal distribution \(q_d(\theta_d|\theta_d^{(j-1)})\)

  3. If \(j=N\), stop, otherwise put \(j\) to \(j+1\), and return to step 2.

Remarks:

  • This is in fact (close to) the original form of the Metropolis algorithm.

  • If the FCD \(\pi(\theta_i|\boldsymbol{\theta}_{-i})\) is available for sampling from directly for a particular component \(\theta_{i}\), it is easy to see that the resulting acceptance probability is 1. When all FCDs are available for sampling from, we obtain an algorithm known as the Gibbs sampler.

3.2 The Gibbs sampler

We will consider first the algorithm and then examine its properties. The algorithm has the following form.

  1. Initialise the state of the chain to \(\boldsymbol{\theta}^{(0)}=(\theta_1^{(0)},\ldots,\theta_d^{(0)})'\). Initialise the iteration counter to \(j=1\).

  2. Obtain a new value \(\boldsymbol{\theta}^{(j)}\) from \(\boldsymbol{\theta}^{(j-1)}\) by successive generation of values

    • \(\theta_1^{(j)}\sim \pi(\theta_1|\theta_{2}^{(j-1)},\ldots,\theta_{d}^{(j-1)})\)

    • \(\theta_2^{(j)}\sim \pi(\theta_2|\theta_{1}^{(j)},\theta_{3}^{(j-1)},\ldots,\theta_{d}^{(j-1)})\)

    • \(\vdots\)

    • \(\theta_d^{(j)}\sim \pi(\theta_d|\theta_{1}^{(j)},\ldots,\theta_{d-1}^{(j)})\)

  3. If \(j=N\), stop, otherwise put \(j\) to \(j+1\), and return to step 2.

Remarks:

  • The algorithm above updates the components of \(\boldsymbol{\theta}\) in order. This is known as the deterministic scan Gibbs sampler.

  • The algorithm clearly defines a homogeneous Markov chain, as each simulated value depends only on the previous simulated value and not on any previous value or the iteration counter \(j\).

3.2.1 Validity

A note on reversible chains

If \(\boldsymbol{\theta}^{(0)},\boldsymbol{\theta}^{(1)},\ldots,\boldsymbol{\theta}^{(N)}\) is a Markov chain then it is straightforward to show that the reversed sequence of states \(\boldsymbol{\theta}^{(N)}, \boldsymbol{\theta}^{(N-1)},\ldots, \boldsymbol{\theta}^{(0)}\) is also a Markov chain. Let \(p(\boldsymbol{\phi}|\boldsymbol{\theta})\) denote the transition density for the forward chain. The reversed chain is not homogeneous in general, however, if the chain is in equilibrium then the transition density for the reversed chain is, via Bayes Theorem \[p^*(\boldsymbol{\phi}|\boldsymbol{\theta}) = \frac{p(\boldsymbol{\theta}|\boldsymbol{\phi})\pi(\boldsymbol{\phi})}{\pi^(\boldsymbol{\theta})}.\]

Definition: The chain is reversible if \(p^*(\boldsymbol{\phi}|\boldsymbol{\theta})=p(\boldsymbol{\phi}|\boldsymbol{\theta})\)

A consequence of reversibility is the detailed balance equation: \[\pi(\boldsymbol{\theta})p(\boldsymbol{\phi}|\boldsymbol{\theta})=\pi(\boldsymbol{\phi})p(\boldsymbol{\theta}|\boldsymbol{\phi}).\] Alternatively, if a chain satisfies detailed balance then it is also reversible. Unfortunately, the deterministic scan Gibbs sampler is not reversible, so we can’t use detailed balance to check that \(\pi(\boldsymbol{\theta})\) is stationary.

Direct checking

The Gibbs sampler admits a Markov chain with transition density \[p(\boldsymbol{\phi}|\boldsymbol{\theta}) = \prod_{i=1}^d\pi(\phi_i | \phi_1,\ldots,\phi_{i-1},\theta_{i+1},\ldots,\theta_d).\] To check stationarity directly, we show that \[\pi(\boldsymbol{\phi}) = \int p(\boldsymbol{\phi}|\boldsymbol{\theta})\pi(\boldsymbol{\theta})d\boldsymbol{\theta}.\] Starting with the RHS, we have \[\begin{aligned} \int p(\boldsymbol{\phi}|\boldsymbol{\theta})\pi(\boldsymbol{\theta})d\boldsymbol{\theta}&= \int\cdots\int \pi(\boldsymbol{\theta})d\theta_1\, \pi(\phi_1|\theta_2,\ldots,\theta_d)\ldots \pi(\phi_d|\phi_1,\ldots,\phi_{d-1})d\theta_2\ldots d\theta_d\\ &=\int\cdots\int \pi(\phi_1,\theta_2,\ldots,\theta_d)d\theta_2\, \pi(\phi_2|\phi_1,\theta_3,\ldots,\theta_d)\ldots \pi(\phi_d|\phi_1,\ldots,\phi_{d-1})d\theta_3\ldots d\theta_d\\ &= \ldots\\ &= \int \pi(\phi_1,\ldots,\phi_{d-1},\theta_d)d\theta_d\, \pi(\phi_d|\phi_1,\ldots,\phi_{d-1})\\ &=\pi(\phi_1,\ldots,\phi_d)\end{aligned}\] as required.

3.2.2 Reversible Gibbs samplers

While the standard deterministic scan version of the Gibbs sampler presented above is not reversible, each component update is in fact reversible. Hence there are several variations of the Gibbs sampler that are reversible, and hence do satisfy the detailed balance equation. Let’s start by considering each component update separately.

Suppose we wish to update component \(i\), that is, we update \(\boldsymbol{\theta}\) by replacing \(\theta_i\) with \(\phi_i\) drawn from \(\pi(\phi_i|\boldsymbol{\theta}_{-i})\), where \(\boldsymbol{\theta}_{-i}=(\theta_{1},\ldots,\theta_{i-1},\theta_{i+1},\ldots,\theta_{d})'\). All the other components are not changed. The transition kernel for this update is \[p(\boldsymbol{\phi}|\boldsymbol{\theta})=\pi(\phi_i|\boldsymbol{\theta}_{-i})I(\boldsymbol{\theta}_{-i}=\boldsymbol{\phi}_{-i})\] where \[I(E)=\begin{cases}1& \textrm{if $E$ is true}\\0 &\textrm{if $E$ is false}\end{cases}.\] Note that the density is zero for a transition changing any but the \(i\)-th component. Now we can check detailed balance: \[\begin{aligned} \pi(\boldsymbol{\theta})p(\boldsymbol{\phi}|\boldsymbol{\theta})&= \pi(\boldsymbol{\theta})\pi(\phi_i|\boldsymbol{\theta}_{-i})I(\boldsymbol{\theta}_{-i}=\boldsymbol{\phi}_{-i})\\ &= \pi(\boldsymbol{\theta}_{-i})\pi(\theta_i|\boldsymbol{\theta}_{-i}) \pi(\phi_i|\boldsymbol{\theta}_{-i})I(\boldsymbol{\theta}_{-i}=\boldsymbol{\phi}_{-i})\\ &= \pi(\boldsymbol{\phi}_{-i})\pi(\theta_i|\boldsymbol{\phi}_{-i}) \pi(\phi_i|\boldsymbol{\phi}_{-i})I(\boldsymbol{\theta}_{-i}=\boldsymbol{\phi}_{-i})\quad\textrm{since\ }\boldsymbol{\phi}_{-i}=\boldsymbol{\theta}_{-i}\\ &= \pi(\boldsymbol{\phi})\pi(\theta_i|\boldsymbol{\phi}_{-i}) I(\boldsymbol{\theta}_{-i}=\boldsymbol{\phi}_{-i})\\ &=\pi(\boldsymbol{\phi})p(\boldsymbol{\theta}|\boldsymbol{\phi}).\end{aligned}\] Detailed balance therefore holds, and hence the update is reversible with stationary distribution \(\pi(\boldsymbol{\theta})\).

Remarks:

  • Recall that the chain defined by a single update is reducible and hence will not converge to the stationary distribution from an arbitrary starting point.

  • In order to ensure irreducibility we need to make sure that we update each component of the chain sufficiently often. As we have seen, one way to do this is to update each component in a fixed order. The drawback with this is that we lose reversibility when we do this.

  • An alternative is to pick a component at random at each stage, and update that. This gives a reversible chain with the required stationary distribution, and is known as the random scan Gibbs sampler.

3.2.3 Gibbs sampling in practice

Example 3.2.1: (Bivariate Gaussian target.) Consider a bivariate normal target with zero mean and unit variance for the marginals, but a correlation of \(\rho\) between the two components. We have that \[\begin{pmatrix} \theta_1\\ \theta_2 \end{pmatrix}\sim N\left\{\begin{pmatrix}0\\0\end{pmatrix}\,,\, \begin{pmatrix} 1 & \rho \\ \rho & 1 \end{pmatrix}\right\}\] with associated density \[\pi(\theta_{1},\theta_{2})\propto \exp\left\{-\frac{1}{2(1-\rho^2)}\left(\theta_{1}^{2}+\theta_{2}^{2}-2\rho \theta_{1}\theta_{2}\right)\right\},\quad -\infty <\theta_1,\theta_2<\infty.\] Let us construct a Gibbs sampler for this target.

The full conditional density for \(\theta_1\) is \[\begin{aligned} \pi(\theta_{1}|\theta_{2})&=\frac{\pi(\theta_{1},\theta_{2})}{\pi(\theta_2)}\\ &\propto \pi(\theta_{1},\theta_{2})\\ &\propto \exp\left\{-\frac{1}{2(1-\rho^2)}\left(\theta_{1}^{2}-2\rho \theta_{1}\theta_{2}\right)\right\}\\ &\propto \exp\left\{-\frac{1}{2(1-\rho^2)}\left(\theta_{1}-\rho \theta_{2}\right)^2\right\}\end{aligned}\] for which we find the full conditional distribution for \(\theta_1\) as \[\theta_{1}|\theta_{2}\sim N\left(\rho \theta_{2},1-\rho^2\right).\] By symmetry, the full conditional distribution for \(\theta_2\) is \[\theta_{2}|\theta_{1}\sim N\left(\rho \theta_{1},1-\rho^2\right)\,.\] The Gibbs sampler for the target above then has the following form.

  1. Initialise the state of the chain to \(\boldsymbol{\theta}^{(0)}=(\theta_1^{(0)},\theta_2^{(0)})'\). Initialise the iteration counter to \(j=1\).

  2. Obtain a new value \(\boldsymbol{\theta}^{(j)}\) from \(\boldsymbol{\theta}^{(j-1)}\) by successive generation of values

    • \(\theta_1^{(j)}\sim N\left(\rho \theta_{2}^{(j-1)},1-\rho^2\right)\)

    • \(\theta_2^{(j)}\sim N\left(\rho \theta_{1}^{(j)},1-\rho^2\right)\)

  3. If \(j=N\), stop, otherwise put \(j\) to \(j+1\), and return to step 2.

The following R code implements this sampler.

## Example: Gibbs sampler for a bivariate normal
 
 gibbs=function(N,rho) 
 {
  mat=matrix(ncol=2,nrow=N)
  th1=0
  th2=0
  mat[1, ]=c(th1,th2)
  for (i in 2:N) 
  {
   th1=rnorm(1,rho*th2,sqrt(1-rho^2))
   th2=rnorm(1,rho*th1,sqrt(1-rho^2))
   mat[i, ]=c(th1,th2)
  }
  return(mat)
 }
 
 out=gibbs(1000,0.5)
 par(mfrow=c(3,2))
 plot(out,col=1:1000,xlab="theta1",ylab="theta2")
 plot(out,type="l",xlab="theta1",ylab="theta2")
 plot(ts(out[,1]),xlab="Iteration",ylab="theta1")
 plot(ts(out[,2]),xlab="Iteration",ylab="theta2")
 hist(out[,1],40,freq=FALSE,main="",xlab="theta1")
 hist(out[,2],40,freq=FALSE,main="",xlab="theta2")
